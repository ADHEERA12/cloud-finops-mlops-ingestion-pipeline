# Cloud Infrastructure FinOps & MLOps Ingestion Pipeline

## Project Goal

Build a production-grade cloud infrastructure intelligence platform that:

- Collects telemetry metrics
- Processes streaming infrastructure data
- Predicts cloud infrastructure costs
- Serves predictions through APIs
- Supports monitoring and MLOps workflows

- FastAPI

- Kafka
